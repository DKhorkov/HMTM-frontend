## Getting Started

To run application in dev mode, you'll need to install **[pnpm](https://pnpm.io/installation)**

```bash
cp .env.example .env.local
Fill env variables
```

#### Dev mode:

```bash
task dev -v
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

#### Build with Docker:

```yaml
task start -v
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
